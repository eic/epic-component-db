### The Component Database (CDB)
---

#### About

The Component Database (CDB) is a web-based tool developed at Argonne National Laboratory to document, organize, and track components used in the MBA accelerator. It serves as a centralized repository connecting component documentation, physical inventory, and machine design planning.

The original source code is managed in a [GitHub Repository](https://github.com/AdvancedPhotonSource/ComponentDB){:target="_blank"}.
For introductory information about the system, please see the [_Summary_](_pages/summary.md) link on top. It
is based on the User Guide created by the authors of this software and on usage notes written by the BNL personnel
testing the system.

The [_Design_](_pages/design.md) page contains information on _designs_, which are aggregations/assemblies of inventory items, as explained in the Summary.
